# baseline-kali

Kali Linux base container with core security tools.

## Usage

```bash
baseline-kali              # Interactive zsh shell
baseline-kali <command>    # Run specific command
```

## What's Inside

- Kali Linux rolling
- kali-linux-core metapackage
- zsh shell

## Notes

Extend with specific tool packages as needed (e.g., `kali-linux-web`, `kali-tools-wireless`).
